Что такое XRat?

XRat - это высокоэффективное приложение для удаленного управления, использующее Discord.

Особенности

  • Только windows 7+
  • Удобный интерфейс
  • Встроенные функции безопасности
  • Ратка управляется через сервер Discord.

Как установить

Чтобы установить XRat, просто скачайте последнюю версию с нашего официального репозитория и следуйте инструкциям по установке.
